Warning

There is a known security issue with the validation of filter parameters in Open XDMoD {{ page.version }}. We highly recommend upgrading to Open XDMoD {{site.latest_sw_version}}.

If upgrading to the latest supported version is not an option, then your Open XDMoD instance should be patched as follows:

RPM installs

  1. Download the updated source file GroupBy.php
  2. Copy this file to: /usr/share/xdmod/classes/{{ include.gbpath }}

Source code installs

  1. Download the updated source file GroupBy.php
  2. Copy this file to: /opt/xdmod/share/classes/{{ include.gbpath }} (where /opt/xdmod is the installation prefix)
